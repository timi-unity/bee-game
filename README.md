# ants vs bees – Tower Defense System Prototype

## Sobre el proyecto
Prototipo de tower defense en 2D desarrollado en Unity (C#), enfocado en la creación de sistemas modulares y escalables.

El objetivo principal fue construir una base flexible para manejar unidades, economía y progresión de dificultad sin depender de valores hardcodeados.

---

## Features

- Sistema modular de tropas con comportamientos independientes  
- UI dinámica: menú individual para mejoras por unidad  
- Sistema de economía por oleadas  
- Escalado progresivo de dificultad parametrizado  
- Diferentes roles de unidades (daño, control, economía)  

---

## Unidades

- **Abeja** → daño por segundo (DPS)  
- **Araña** → ralentización de enemigos  
- **Escarabajo** → generación de recursos por oleada  

Cada unidad cumple un rol específico dentro de la estrategia del jugador.

---

## Enfoque técnico

- Arquitectura modular para facilitar la escalabilidad de unidades  
- Separación de lógica por tipo de comportamiento  
- Sistema de progresión basado en parámetros (sin hardcoding)  
- UI vinculada dinámicamente a entidades del juego  

---

## Qué aprendí

- Separar responsabilidades en la lógica del juego

- Construir progresión dinámica basada en variables
